aula 4 - conceitos

linguagem de programação

uma "língua" que tanto você (programador) quanto o computador entendem.

exemplos:

  • Python
  • JavaScript
  • PHP

IDE: ambiente de desenvolvimento integrado

costuma ter:

  • editor de texto puro
  • algo que executa seu código (compilador ou integrador)
  • navegador de arquivos

exemplos:

  • Google Colab
  • Visual Studio
  • Eclipse

e o HTML?

o HTML não é uma linguagem de programação, mas...

  • é uma linaguagem em texto puro que você e o computador entendem
  • só que não é de programação, é uma linguagem de descrição de documentos
  • o navegador de internet lê o código HTML e monta uma página
